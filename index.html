HTML Nedir?

HTML (HyperText Markup Language) web sayfalarının iskeletidir. İçeriği tanımlar ama görünümü değil. Görünüm CSS'in işi.

HTML — Temel Sayfa Yapısı 
<!DOCTYPE html>          ← Tarayıcıya "HTML5 kullanıyorum" der
<html lang="tr">            ← Kök eleman, dil belirt
  <head>                      ← Görünmez meta bilgiler
    <meta charset="UTF-8">
    <title>Sayfam</title>
  </head>
  <body>                      ← Görünen her şey burada
    <h1>Merhaba Dünya!</h1>
  </body>
</html>
Altın Kural: HTML = yapı, CSS = görünüm, JavaScript = davranış. Bu ayrımı asla karıştırma.

Semantic HTML

Semantic = anlamlı. Her şeyi div yapmak yerine, içeriği tanımlayan etiketleri kullanmak hem okunabilirliği hem SEO'yu iyileştirir.

❌ Kötü (amatör):
Her şey div
HTML — Yanlış
<div class="header">...</div>
<div class="nav">...</div>
<div class="content">...</div>
<div class="footer">...</div>
✅ İyi (pro):
Anlamlı etiketler
HTML — Doğru
<header>...</header>
<nav>...</nav>
<main>...</main>
<footer>...</footer>

Semantic Etiket Haritası

<header> — Logo, site başlığı, üst menü
<nav> — Navigasyon linkleri
<main>
<section> — Bölümler
<article> — Bağımsız içerik
<aside>
Yan panel
<footer> — Alt bilgi, telif, iletişim
Sınav Trick: Hoca semantic etiket kullanımını kontrol eder. <header>, <nav>, <main>, <footer> kullan → puan garantili.

CSS Temelleri

CSS Nedir?

CSS (Cascading Style Sheets) HTML iskeletine görünüm kazandırır. Renk, boyut, konum, animasyon — hepsi CSS.

CSS Nasıl Eklenir?

❌ INLINE — kullanma
<p style="color:red">Metin</p>
⚠️ INTERNAL — küçük projeler
<style>
  p { color: red; }
</style>
✅ EXTERNAL — her zaman bu
<link rel="stylesheet" href="style.css">

CSS Sözdizimi

CSS
selector {         ← Kimi hedefliyoruz?
  property: value;  ← Ne değişiyor?
}

/* Örnek: */
p {
  color: red;
  font-size: 16px;
}

.kart {            ← class seçici
  background: #fff;
}

#logo {            ← id seçici
  width: 200px;
}
CSS Temelleri

Selector'lar

CSS hangi HTML elemanını hedefleyeceğini selector'larla bilir.

CSS — Temel Selector'lar
p { }               ← Tüm p etiketleri
.kart { }          ← class="kart" olan elemanlar
#logo { }          ← id="logo" olan TEK eleman
* { }               ← Her şey (evrensel)

/* Kombinasyonlar */
div p { }          ← div içindeki TÜM p'ler
div > p { }        ← div'in DOĞRUDAN çocuğu p
h1, h2 { }         ← h1 VE h2 (virgül = liste)

/* Pseudo-class (durum seçici) */
a:hover { }        ← Üzerine gelince
li:first-child { } ← İlk li
li:nth-child(2) { }← 2. li
p:not(.ozel) { }  ← .ozel sınıfı olmayan p'ler

/* Attribute Selector */
[type="text"] { }  ← type="text" olan input
Kural: id = sayfada tek olmalı → #id. class = birden fazla eleman paylaşabilir → .class

Pseudo Elements

CSS
p::before {
  content: "→ ";    ← p'nin ÖNÜNE ekle
  color: purple;
}

p::after {
  content: " ✓";   ← p'nin SONUNA ekle
}
CSS Temelleri

Box Model

CSS'de her eleman bir kutudur. Bu kutunun 4 katmanı var.

content
padding
İçerik ile border arasındaki boşluk. İç boşluk.
border
Kutunun çerçevesi. Kalınlık, renk, stil belirlenebilir.
margin
Diğer elemanlarla arasındaki boşluk. Dış boşluk.
content
Asıl içerik. width/height burayı etkiler.
CSS — Box Model Kullanımı
.kutu {
  width: 200px;
  height: 100px;
  padding: 20px;         ← 4 taraf aynı
  padding: 10px 20px;    ← üst-alt / sağ-sol
  border: 2px solid red;
  margin: 16px;
  
  /* ALTIN KURAL → layout kırılmaz */
  box-sizing: border-box;
}
Dikkat: Normal halde width: 200px + padding: 20px = toplam 240px eder. box-sizing: border-box ile padding width'in içine dahil olur → toplam yine 200px.
CSS Reset'e ekle: * { box-sizing: border-box; } → bütün projeyi kurtarır.

Display & Position

Display Değerleri

CSS
display: block;        ← Tam satır kaplar (div, p, h1...)
display: inline;       ← İçerik kadar yer (span, a...)
display: inline-block; ← Inline + width/height ayarlanabilir
display: none;         ← Tamamen gizle (yer kaplamaz)
display: flex;         ← Flexbox (bir sonraki konu!)
display: grid;         ← Grid layout

Position Değerleri

CSS
position: static;    ← Varsayılan, akışta normal yer
position: relative;  ← Kendi konumuna göre offset
position: absolute;  ← En yakın relative parent'a göre
position: fixed;     ← Ekrana yapışır (sticky navbar)
position: sticky;    ← Scroll'a kadar relative, sonra fixed

/* Offset özellikleri: */
top: 20px;
left: 50px;
right: 0;
bottom: 10px;
Kritik Kural: position: absolute kullanan eleman, üst elemanının position: relative olmasını bekler. Aksi halde sayfanın köküne göre konumlanır.
CSS — Absolute içinde Relative Pattern
.kart {
  position: relative;  ← Parent relative olmalı
}

.rozet {
  position: absolute;
  top: 10px;
  right: 10px;           ← .kart'ın sağ üstüne yapışır
}

Flexbox

CSS'in en güçlü layout aracı. Elemanları satır veya sütun halinde hizalamayı kolaylaştırır.

CSS — Flexbox Temeli
.container {
  display: flex;
  
  /* Yön */
  flex-direction: row;         ← yatay (varsayılan)
  flex-direction: column;      ← dikey
  
  /* Yatay hizalama */
  justify-content: flex-start; ← sola
  justify-content: center;     ← ortaya
  justify-content: flex-end;   ← sağa
  justify-content: space-between; ← araya boşluk
  justify-content: space-around;  ← eşit boşluk
  
  /* Dikey hizalama */
  align-items: center;        ← dikeyde ortaya
  align-items: flex-start;    ← üste
  align-items: stretch;       ← uzat (varsayılan)
  
  gap: 16px;               ← elemanlar arası boşluk
  flex-wrap: wrap;          ← sığmazsa alta geç
}

Layout

CSS Grid

Flex tek eksen (yatay veya dikey), Grid ise iki eksen aynı anda. Kompleks layoutlar için idealdir.

CSS — Grid Temeli
.grid {
  display: grid;
  
  /* Sütun tanımı */
  grid-template-columns: 200px 200px 200px; ← 3 sütun
  grid-template-columns: 1fr 1fr 1fr;       ← 3 eşit sütun
  grid-template-columns: repeat(3, 1fr);   ← aynı şey, kısa yol
  grid-template-columns: 2fr 1fr;          ← 2:1 oran
  
  /* Otomatik responsive */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  
  gap: 20px;            ← grid boşlukları
}

Responsive Tasarım

Her ekran boyutunda düzgün görünen site. Media query ile farklı ekranlara farklı CSS yazılır.

CSS — Media Query
/* Varsayılan: masaüstü */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

/* Tablet: 768px altında */
@media (max-width: 768px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Mobil: 480px altında */
@media (max-width: 480px) {
  .container {
    grid-template-columns: 1fr;  ← tek sütun
  }
}

Yaygın Breakpoint'ler

📱
≤ 480px
Mobil
📱
≤ 768px
Tablet
💻
≤ 1024px
Laptop
🖥️
> 1200px
Masaüstü
Mobile-First Yaklaşım (Pro): Önce mobil CSS yaz, sonra büyük ekranlar için min-width ile genişlet. Amatör → büyükten küçüğe. Pro → küçükten büyüğe.

CSS Specificity

Birden fazla CSS kuralı aynı elemanı hedeflediğinde, hangisi kazanacak? Bunu specificity (özgüllük) belirler.

Kazanma Sıralaması

#id { }
100 puan
.class { }
10 puan
p { }
1 puan
* { }
0 puan
CSS — Kim Kazanır?
p { color: blue; }          ← specificity: 1
.metin { color: red; }    ← specificity: 10 → KAZANIR
#baslik { color: green; } ← specificity: 100 → EN GÜÇLÜ

/* Kombine örnek: */
div.kart p { }  ← 0+10+1+1 = 12 puan
#nav .link { }  ← 100+10 = 110 puan → kazanır
!important Kullanma: color: red !important; her şeyi ezer ama bir hata işareti. "Specificity anlamıyorum" demek gibi. Pro bu işe başvurmaz.

CSS Variables

Değerleri bir kez tanımla, her yerde kullan. Rengi değiştirmek istersen tek yerden değiştir.

CSS — Custom Properties
/* :root = tüm sayfada geçerli */
:root {
  --primary: #7c6aff;
  --secondary: #ff6a9b;
  --text: #333;
  --radius: 12px;
  --spacing: 16px;
}

/* Kullanım: */
.buton {
  background: var(--primary);
  border-radius: var(--radius);
  padding: var(--spacing);
}

.buton-ikincil {
  background: var(--secondary);
  border-radius: var(--radius);  ← Aynı radius, tek değişiklik
}
Neden Kullanmalı? Renk temasını değiştirmek istersen tek yerden değiştirirsin. Büyük projelerde hayat kurtarır.

BEM Mimarisi

BEM = Block Element Modifier. CSS class isimlerini sistematik yazmak için bir yöntem.

CSS — BEM Yapısı
/* BLOCK — bağımsız bileşen */
.kart { }

/* ELEMENT — bloğun parçası (__) */
.kart__baslik { }
.kart__resim { }
.kart__icerik { }

/* MODIFIER — varyasyon (--) */
.kart--buyuk { }
.kart--aktif { }
.kart--disabled { }
HTML — BEM Kullanımı
<div class="kart kart--buyuk">
  <img class="kart__resim" src="...">
  <h2 class="kart__baslik">Başlık</h2>
  <p class="kart__icerik">İçerik</p>
</div>
❌ Amatör:
.kart-buyuk-baslik
.red-button-hover
.mystylee123
✅ BEM ile:
.kart--buyuk
.buton--kirmizi
.buton__ikon
Özet: BEM kullandığında koduna bakan biri "Bu kart bloğunun başlık elementi" diye anlıyor. Class'tan yapıyı okuyabiliyor. Gerçek pro seviyesi bu.
